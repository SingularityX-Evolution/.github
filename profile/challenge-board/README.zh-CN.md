对于具有竞争性的悬赏挑战，默认建议通过私密邮箱提交；公开 Pull Request 仅适用于贡献者主动选择公开代码、报告和实现细节的情况。

## 难度与赏金说明

难度与人民币赏金上限按完整挑战维度设置，不是子任务报价、阶段付款或保证付款。实际赏金认定以提交材料、复现结果、评审结论、贡献证据、适用协议、合规要求和赏金评审流程为准。

## 提交方式

私密提交发送至：

```text
join@singularityx.tech
```

建议邮件标题：

```text
[Challenge Submission] <Challenge ID> - <Contributor Name or GitHub Username>
```

有效提交应包含可复现代码或结构化交付包、README 与运行说明、测试结果或评估报告、依赖与环境信息、已知限制，以及贡献者确认其拥有或有权提交相关成果的声明。

提交内容一经提交不得更改；任务截止后三个自然日内，主办方会统一给出答复。
