# 因果链标注任务说明

> 这是给帮手使用的标注任务。标注员无需了解代码，只需按照本说明完成 50 条因果链的标注。

## 任务目标

为因果链多跳推理项目标注 50 条金融领域因果链，用于校准置信度聚合模型。

## 什么是因果链

因果链是由多个因果跳跃（hop）组成的推理链条，例如：

```
美联储降息  →  实际利率下行  →  成长股估值提升  →  股价上涨
   hop 1          hop 2            hop 3
```

## 每条链需要填写的信息

对于每条链，请填写以下内容：

### 1. chain_id（链编号）
格式：`CH_XXX`（如 CH_001, CH_002）

### 2. hops（跳列表）
列出该因果链的每一跳，例如：

```
hops: [
  "美联储宣布降息25bp",
  "商业银行存贷款利率下调",
  "企业融资成本降低",
  "制造业投资增加"
]
```

### 3. domain（领域）
从以下选择一个：
- `macro`：宏观经济（货币政策、财政政策等）
- `commodity`：大宗商品（原油、黄金、农产品等）
- `equity`：股票市场（行业轮动、个股分析等）
- `crypto`：加密货币（比特币、以太坊等）
- `forex`：外汇市场
- `credit`：信用市场

### 4. label（最终结果是否成立）
标注最后一跳的结论在现实中是否实际发生。

格式：直接写 `true` 或 `false`

**判断标准**：
- `true`：最终结论在现实中确实发生了，或者有相当高的概率会发生
- `false`：最终结论在现实中没有发生，或者推理链有严重断裂

### 5. confidence_per_hop（每跳置信度）
对每一跳的因果强度打分，范围 0.0 ~ 1.0：

| 分数 | 含义 |
|---|---|
| 0.95–1.00 | 几乎必然发生，因果关系极强 |
| 0.85–0.94 | 大概率发生，因果关系较强 |
| 0.70–0.84 | 可能发生，因果关系中等 |
| 0.50–0.69 | 不确定，因果关系较弱 |
| < 0.50 | 不太可能，因果链断裂风险高 |

### 6. notes（备注）
记录任何你觉得重要的信息：
- 是否有外部截断事件（OPEC 增产、黑天鹅等）
- 是否有反向因果或混淆因素
- 跳与跳之间是否有时间滞后

## 输出格式

请将标注结果保存为 `annotated_chains.csv`，格式如下：

```
chain_id,hops,domain,label,confidence_per_hop,notes
CH_001,"['事件1', '事件2', '事件3']",macro,true,"[0.95, 0.90, 0.85]","无外部截断"
CH_002,"['事件1', '事件2', '事件3']",commodity,false,"[0.95, 0.80, 0.50]","中间OPEC增产对冲"
...
```

## 注意事项

1. **因果方向**：确保每一跳的因果方向正确（A 导致 B，不是 B 导致 A）
2. **独立性**：尽量选择相互独立的因果链，不同链之间不要重复相同的推理逻辑
3. **覆盖跳数**：尽量覆盖 1 跳、2 跳、3 跳、4 跳的链各 10 条以上
4. **标注一致性**：标注前先通读 5-10 条，感受打分的松紧程度
5. **不要猜测结果**：如果不确定最终结果是否发生，宁可标低置信度，不要猜

## 参考：金融因果链示例

以下是 3 个参考示例，标注员可参照格式：

### 示例 1（macro, 3跳, label=true）

```
chain_id: CH_000a
hops: ["美联储宣布加息50bp", "实际利率上升", "黄金价格下跌"]
domain: macro
label: true
confidence_per_hop: [0.95, 0.88, 0.92]
notes: 经典利率-黄金关系，逻辑清晰
```

### 示例 2（commodity, 2跳, label=false）

```
chain_id: CH_000b
hops: ["中东局势紧张", "原油价格暴涨"]
domain: commodity
label: false
confidence_per_hop: [0.90, 0.60]
notes: 中东紧张但OPEC同时增产，原油未大涨
```

### 示例 3（equity, 4跳, label=true）

```
chain_id: CH_000c
hops: ["俄乌战争爆发", "欧洲能源危机", "化工原料成本上升", "化工板块做空"]
domain: equity
label: true
confidence_per_hop: [0.95, 0.85, 0.80, 0.75]
notes: 4跳链，注意第3跳到第4跳之间可能有政策干预
```

## 验收标准

- 共 50 条因果链
- 覆盖 1-4 跳，每个跳数至少 10 条
- 覆盖至少 4 个不同领域
- 每条链有完整的 hops、label、confidence_per_hop
- CSV 文件格式正确，可被 pandas 读取
