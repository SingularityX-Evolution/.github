## 二、结论

### S 档标准达成

- Brier Score = 0.1236 < 0.15（S 档要求）
- 不同跳数胜率呈现合理的 2跳 > 3跳 > 4跳 衰减趋势
- DampedMultiplier 相比朴素连乘 baseline（Brier=0.1760）提升 29.8%

### 核心发现

1. **朴素连乘的问题在长链上暴露明显**：4跳时朴素连乘胜率仅 0.56（最低），因为多跳乘积使置信度衰减过度，导致很多正例被错误判断为负例。

2. **DampedMultiplier 有效缓解长链衰减**：通过跳序加权调整，早期跳的影响力被合理折扣，4跳胜率维持在 0.76，显著优于 baseline。

3. **LogOdds 和 BayesianUpdater 表现接近**：均优于朴素 baseline，说明对数几率空间聚合和贝叶斯更新都好过独立概率乘法。

4. **NoisyOR 当前参数过于激进**：默认参数导致对所有链都输出极高置信度（0.97+），reliability 0.4578 表明严重校准失败，需要调参。

5. **停止准则**：组合 OR 模式下，CIWidthStopper 是最敏感的触发器——当置信度接近 1.0 时，CI 宽度收窄到 0.01 以下，立即触发停止，避免了过度推理。

## 三、朴素连乘失效分析

### 数学原因

1. **独立性假设失效**：各跳之间存在共同因果因素（如"宏观宽松"同时影响多个跳），朴素乘法假设各跳独立导致重复扣减。

2. **长链过度衰减**：4 跳时，即使每跳置信度高达 0.85，朴素乘积 = 0.85^4 = 0.52，低于 0.5 的判断阈值，将大量正例误判为负例。

3. **无置信区间**：朴素方法不提供不确定性区间，无法判断"置信度 0.52 接近边界"的情况。

4. **外部截断事件未建模**：OPEC 增产、地缘降温等中间事件可能打断链，但朴素方法无法吸收这类信息。

### DampedMultiplier 的改进

- 引入跳序衰减因子：早期跳影响力折扣更多，后期跳保留更多权重
- 公式：`result = ∏ p_i^(w_i) · (1 + α·i/n)`，其中 w_i = 1/(1+δ·i)
- 效果：4 跳时等效乘积因子约为 0.75，相比朴素 0.52 更接近真实

## 五、局限性与后续方向

1. **数据局限**：当前验证基于 50 条合成链，真实标注数据可能表现出不同的跳数分布和领域偏差。

2. **NoisyOR 调参空间**：当前参数使 NoisyOR 过于激进，需要系统调参（grid search on leak_prob, inhibition_base）以达到可接受校准。

3. **外部截断建模**：当前框架支持传入干扰因子，但未在合成数据中验证不同干扰水平的效果。

4. **冷启动问题**：新领域的第一跳置信度估计依赖人工标注，缺乏自动化方法。
